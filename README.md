# LichSOMA's Speaker Stage

> **이 모듈은 생성형 AI의 도움으로 제작되었습니다.**

## 호환성

- **Foundry VTT**: v14 이상

## 소개

[LichSOMA's Speaker Selector](https://github.com/LichSOMA/lichsoma-speaker-selector)를 확장하는 모듈입니다. 채팅과 연동된 **스테이지**(스탠딩·대화창), **백스테이지**, 선택적으로 **스테이지 배경**(16:9)·시간대 연동 등을 제공합니다. Speaker Selector와 함께 활성화해야 합니다.

## 주요 기능

### 스테이지 / 대화창

- 화면 하단 오버레이에 비주얼 노벨 형태로 캐릭터와 대화창 표시.
- 채팅 메시지와 연동된 타이핑 연출·폰트 등은 모듈 설정에서 조정.
- **스테이지 중앙 제어**(월드, GM): 켜 두면 GM이 스테이지 표시를 켜거나 끌 때 플레이어 클라이언트에도 동일하게 반영. 이때 플레이어에게는 스테이지 on/off 버튼을 표시하지 않음.

### 백스테이지

- 챗 폼 스피커 셀렉터 아래에 등록 스피커 액터 포트레잇 목록 표시(GM).
- 전구 버튼으로 스테이지 표시 on/off; 포트레잇 클릭으로 스테이지 올리기·스피커 선택.

### 등록 스피커와 스테이지

- 스피커 셀렉터의 등록 목록(`actorGridActors`)과 동일 기준으로 동작. 등록에서 빠진 액터는 스테이지에서도 자동으로 제거됨.

### 스테이지 배경

- 현재 장면과 무관하게 스테이지 영역에 배경 이미지 표시.
- **설정**: 게임 설정 → 모듈 설정의 「스테이지 배경 장면」(GM)에서 슬롯(최대 99)에 이미지·이름 등록 후 표시 슬롯 선택.
- **미니 모드**(클라이언트): 작은 16:9 프레임으로만 표시.
- **내러티브 모드**(월드): 배경을 토큰 위에 그리며 캔버스 조작을 차단. 변경 시 세션 리로드 안내가 표시됩니다.

### 시간대 연동(선택)

- 월드 설정 **시간대 연동**을 켜면 슬롯별로 **오전 / 오후 / 심야** 이미지를 구분해 쓸 수 있음. 변경 시 세션 리로드 안내가 표시됩니다.
- 사용하는 **시간대**는 `timeOfDay` 월드 설정 값입니다. **LichSOMA's Daily Calendar** (`lichsoma-daily-calendar`)를 켜 두면 그 모듈의 시간대와 연동됩니다.
- Daily Calendar가 없거나 비활성이면 연동 옵션만으로는 게임 내 시간대가 바뀌지 않습니다.

## 설치

GitHub 릴리스의 `module.json` manifest URL로 설치하거나, 모듈 폴더를 `Data/modules` 아래에 두고 활성화합니다.
