# LichSOMA's Speaker Stage

LichSOMA's Speaker Selector 모듈을 확장하여 비주얼노벨 스타일의 캐릭터 스탠딩과 대화창을 지원하는 Foundry VTT 모듈입니다.

## 필수 모듈

- [LichSOMA's Speaker Selector](https://github.com/yourusername/lichsoma-speaker-selecter)

## 기능

### 백스테이지 (Backstage)

챗 폼에 등록된 스피커 액터들의 포트레잇이 표시되는 백스테이지 UI를 제공합니다.

- **액터 포트레잇 목록**: 스피커 셀렉터에 등록된 액터들의 포트레잇을 표시
- **토글 버튼**: 백스테이지 기능을 켜고 끄는 전구 아이콘 버튼

## 설치

1. Foundry VTT의 모듈 폴더에 이 모듈을 복사합니다.
2. Foundry VTT를 시작하고 모듈 관리에서 "LichSOMA's Speaker Stage"를 활성화합니다.
3. "LichSOMA's Speaker Selector" 모듈도 함께 활성화되어 있어야 합니다.

## 사용법

모듈을 활성화하면 챗 폼의 스피커 셀렉터 바로 아래에 백스테이지 UI가 표시됩니다.
